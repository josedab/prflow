## [Unreleased]

### Added
- Initial open source release
- Multi-agent PR analysis system with Analyzer, Reviewer, Test Generator, and Documentation agents
- GitHub App integration for automated PR reviews
- GitHub Action for CI/CD pipeline integration
- Web dashboard for PR analytics and team insights
- VS Code extension for pre-commit analysis
- Learning system that improves from developer feedback
- Enterprise features: SSO, compliance, and team management
- Merge queue with dependency detection

### Security
- JWT-based authentication with secure token handling
- Rate limiting and request validation
- Built-in security scanning for hardcoded secrets
- SQL injection and XSS detection in code reviews
